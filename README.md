# Otaku Inventory

「この素晴らしい世界に祝福を！」関連グッズを整理して表示する個人運営プロジェクトです。

## 主な機能

- 一覧表示（検索 / フィルタ / ページング）
- 在庫状態表示
- お気に入り
- お知らせ・運用ポリシー表示

## 技術スタック

- Frontend: React + Vite + React Router
- Backend: Fastify + better-sqlite3
- DB: SQLite (`goods.db`)
