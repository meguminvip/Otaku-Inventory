# Otaku Inventory

Otaku Inventory は、この素晴らしい世界に祝福を！関連のグッズを追跡・整理をし、見やすい形でユーザーに提供するためのサービスです。

## Requirements

Node.js 18以上
npm

## Setup

### Backend

```
npm install
node server.js
```

### Frontend (Production)

```
npm install
npm run build
npm start
```

### Frontend (Development)

```
npm install
npm run dev
```
